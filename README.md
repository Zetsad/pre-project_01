# หัวข้อ พยากรณ์น้ำท่วม

=====ทำเพื่อใคร===== 
1.ประชากรในพื้นที่ที่เสี่ยงน้ำท่วม
2.เกษตกร ที่ได้รับผลกระทบ
3.พื้นที่ที่เกิดน้ำท่วมซ้ำและบ่อย
4.ประชากรที่ได้รับความเสียหายจากน้ำท่วมและต้องการความช่วยเหลือ

=====สื่งที่คาดหวัง=====
1.สามารถพยากรณ์น้ำท่วมได้อย่างแม่นยำ
2.ติดตามระดับน้ำแบบ realtime 
3.แจ้งเตือนประชากรในพื้นที่ได้ก่อนเหตุการณ์
4.สามาถส่งสัญญาณความช่วยเหลือไปยังหน่วยงานของภาครัฐได้
5.แสดงพื้นที่ลักษณะพื้นที่ภูมิประเทศแต่ละจังหวัดได้ เพื่อที่จะนำไปวิเคราะห์ความเสี่ยงเกิดน้ำท่วม
