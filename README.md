# สืบค้นเอกสารที่เกี่ยวข้องกับโครงงาน

จากสัปดาห์ก่อนที่ได้วางแผนดำเนินงานผมได้สืบค้นข้อมูลและสิ่งที่ต้องรู้เพื่อที่จะนำไปประยุกต์ใช้กับโครงงานดังนี้ ระบบแผนที่(GIS),คู่มือการเตรียมความพร้อมเบื้องต้นเมื่อเกิดภัยพิบัติต่างๆ
(Disaster Management),ระบบนำทาง (Navigation),ระบบใช้วิเคราะห์เส้นทางโดยใช้(AI)
Machine Learningโดยจะใช้เสริม AI เพื่อให้รู้ว่าเส้นทางไหนใกล้ที่สุดและควรเลี่ยงซึ่งได้ข้อมูลดังนี้


# ระบบแผนที่(GIS)
**GIS (Geographic Information System)** หรือระบบสารสนเทศภูมิศาสตร์ คือเทคโนโลยีที่ผสานแผนที่เข้ากับฐานข้อมูลคอมพิวเตอร์ ทำหน้าที่รวบรวม จัดเก็บ วิเคราะห์ และแสดงผลข้อมูลที่มีพิกัดตำแหน่งบนพื้นโลก เพื่อช่วยในการตัดสินใจ วางแผน และแก้ปัญหาเชิงพื้นที่ได้อย่างแม่นยำ

**GIS ใช้ทำไม**
1.เก็บข้อมูลเชิงพื้นที่ (Spatial Data)
2.วิเคราะห์ข้อมูล
3.แสดงผลบนแผนที่
4.ช่วยตัดสินใจจากข้อมูลตำแหน่ง

**ข้อมูลเชิงพื้นที่ (Spatial Data)**
ข้อมูลเชิงพื้นที่นั้น หลัก ๆ มี 2 ประเภทคือ แบบเวกเตอร์ (Vector) และแบบราสเตอร์ (Raster) ข้อมูลเชิงพื้นที่แบบเวกเตอร์ จะแบ่งเป็นประเภทย่อยหลัก ๆ ได้ 3 ลักษณะ ได้แก่ ข้อมูลแบบจุด (Point) ข้อมูลเส้น (Line) และข้อมูลพื้นที่รูปปิด (Polygon หรือ Area)

โดยทั่วไปแล้วเราจะใช้มาตรฐานข้อมูลเชิงพื้นที่ของ ESRI โดยเป็นการจัดเก็บข้อมูลแบบเวกเตอร์ ซึ่งจะได้อธิบายรายละเอียดต่อไป ส่วนข้อมูลเชิงพื้นที่แบบราสเตอร์ผู้เขียนขอยังไม่ได้กล่าวในบทความนี้ การเก็บข้อมูลแบบราสเตอร์นั้น จะเป็นการเก็บข้อมูลแบบพิกเซล (Pixel) โดยรูปภาพข้างล่างนี้ได้แสดงตัวอย่างลักษณะข้อมูลเชิงพื้นที่ทั้งสองแบบ เพื่อให้ท่านเห็นภาพข้อมูลแบบ แบบเวกเตอร์และแบบราสเตอร์(ที่มา [https://geodacenter.github.io/opioid-environment-toolkit/spatial-data-introduction.html](https://geodacenter.github.io/opioid-environment-toolkit/spatial-data-introduction.html) )

สรุปแล้ว**ระบบแผนที่หรือGIS**ที่นำมาประยุกต์ใช้กับโครงงานหลักๆคือ ใช้วิเคราะห์ข้อมูลบนแผนที่ว่าจุดนี้น้ำท่วมถนนเส้นนี้ควรเลี่ยงเพราะไม่สามารถผ่านได้และบอกได้ว่าถนนเส้นไหนที่สามารถใช้งานได้(ที่มา [https://www.esri.com/en-us/what-is-gis/overview]((https://www.esri.com/en-us/what-is-gis/overview)))

# ระบบนำทาง (Navigation)
**ระบบนำทาง (Navigation)** คือ เทคโนโลยีที่ใช้กำหนดตำแหน่งบนโลกและคำนวณเส้นทางเพื่อเดินทางไปยังจุดหมายอย่างแม่นยำ โดยอาศัยการทำงานร่วมกันระหว่างดาวเทียม และแผนที่ดิจิทัล เพื่ออำนวยความสะดวกในการเดินทางทั้งทางบก ทางน้ำ และทางอากาศ
ระบบนำทางในที่นี้จะแตกต่างจาก GPS ตรงที่ GPS ใช้บอกว่าตอนนี้เราอยู๋ไหนโดยรับสัญญาณจากดาวเทียมเพื่อระบุตำแหน่ง ส่วนระบบนำทาง (Navigation) จะบอกเราว่าเราควรใช้เส้นทางไหนที่จะถึงจุดหมายได้ไวโดยการทำงานขอบระบบนำทางจะใช้ตำแหน่งจาก GPS ใช้ข้อมูลบนแผนที่เพื่อคำนวณเส้นทาง(ที่มา [https://eyefleet.co/compare-gps-tracking-navigation-geofencing/](https://eyefleet.co/compare-gps-tracking-navigation-geofencing/))

สรุป **ระบบนำทาง(์Navigation)**ใช้เพื่อคำนวณและแนะนำเส้นทางอพยพที่ปลอดภัยและเหมาะสมที่สุดแก่ผู้ใช้งานในสถานการณ์ฉุกเฉิน โดยสามารถปรับเปลี่ยนเส้นทางตามสถานการณ์ได้แบบเรียลไทม์

# การจัดการภัยพิบัติ (Disaster Management)

**การจัดการภัยพิบัติ (Disaster Management)**คือระบบบริหารจัดการทรัพยากรและมาตรการเพื่อลดความสูญเสียทั้งชีวิตและทรัพย์สิน โดยแบ่งการทำงานออกเป็น 4 ระยะ ได้แก่ การลดผลกระทบ (Mitigation), การเตรียมความพร้อม (Preparedness), การรับมือ (Response), และการฟื้นฟู (Recovery) เพื่อจัดการวิกฤตอย่างเป็นระบบ

**การป้องกันและลดความเสี่ยง (Prevention / Mitigation)**
เป็นการดำเนินการเพื่อลดโอกาสเกิดภัยพิบัติหรือบรรเทาความรุนแรงของผลกระทบ
**การเตรียมความพร้อม (Preparedness)**
เป็นการเตรียมความพร้อมก่อนเกิดเหตุ เพื่อให้สามารถรับมือได้อย่างรวดเร็ว
**การตอบสนองต่อเหตุการณ์ (Response)**
เป็นการดำเนินงานเมื่อภัยพิบัติเกิดขึ้น เพื่อช่วยเหลือผู้ประสบภัยและลดความสูญเสีย
**การฟื้นฟู (Recovery)**
เป็นการฟื้นฟูหลังเหตุการณ์ เพื่อให้ชุมชนกลับมาใช้ชีวิตได้ตามปกติ

**การจัดการภัยพิบัติ (Disaster Management)** เกี่ยวข้องกับโครงงานยังไง
Disaster Management เปรียบเสมือน **แนวทางหรือหลักการ** ส่วนระบบของคุณคือ **เครื่องมือที่ช่วยให้การจัดการภัยพิบัติมีประสิทธิภาพมากขึ้น**

สรุป**Disaster Management** คือหลักการในการจัดการภัยพิบัติอย่างเป็นระบบ ตั้งแต่ก่อนเกิดเหตุ ขณะเกิดเหตุ และหลังเกิดเหตุ ซึ่งโครงงานของเรานำหลักการนี้มาใช้ในการออกแบบระบบนำทางและช่วยเหลือผู้ประสบภัยให้มีประสิทธิภาพมากขึ้น

## ระบบใช้วิเคราะห์เส้นทางโดยใช้(AI)
โดยระบบวิเคราะห์เส้นทางจะใช้ในส่วนของMachine Learning (ML)
**Machine Learning** คือแขนงหนึ่งของ **Artificial Intelligence (AI)** ที่ทำให้ระบบสามารถ **เรียนรู้จากข้อมูลในอดีตและข้อมูลปัจจุบัน** เพื่อนำมาคาดการณ์หรือช่วยตัดสินใจได้ โดยไม่ต้องเขียนกฎทุกอย่างไว้ล่วงหน้า
### นำมาใช้ทำอะไรในโครงงาน
ในแพลตฟอร์มแผนที่นำทางอัจฉริยะเพื่อการอพยพและช่วยเหลือผู้ประสบภัย Machine Learning สามารถนำมาใช้เพื่อคาดการณ์ความเสี่ยงของเส้นทาง,วิเคราะห์รูปแบบการเกิดภัยพิบัติ,
แนะนำเส้นทางล่วงหน้าปรับปรุงความแม่นยำของระบบ

สรุป**Machine Learning** คือเทคโนโลยีที่ทำให้ระบบเรียนรู้จากข้อมูล เพื่อคาดการณ์สถานการณ์และช่วยตัดสินใจได้อย่างแม่นยำมากขึ้น เช่น การทำนายพื้นที่เสี่ยงหรือเส้นทางที่อาจได้รับผลกระทบจากภัยพิบัติ ก่อนที่จะเกิดเหตุจริง(ที่มา[https://www.disruptignite.com/blog/machine-learning](https://www.disruptignite.com/blog/machine-learning))


## Artificial Intelligence (AI)

**Artificial Intelligence (AI)** คือ เทคโนโลยีที่ทำให้คอมพิวเตอร์สามารถ **วิเคราะห์ข้อมูล เรียนรู้ ตัดสินใจ และให้คำแนะนำ** ได้คล้ายกับการใช้เหตุผลของมนุษย์ โดยอาศัยข้อมูลที่ได้รับจากแหล่งต่าง ๆ
### AI เกี่ยวข้องกับโครงงานอย่างไร
 AI มีหน้าที่ **วิเคราะห์ข้อมูลจากหลายแหล่ง** เช่น GPS, GIS, ข้อมูลสภาพอากาศ, ระดับน้ำ, ถนนปิด และข้อมูลการจราจร เพื่อช่วยตัดสินใจและแนะนำเส้นทางอพยพที่ปลอดภัยที่สุดให้แก่ผู้ใช้งานและยังช่วยวิเคราะห์ระดับความเสี่ยงของแต่ละเส้นทางแนะนำเส้นทางอพยพที่ปลอดภัยที่สุดปรับเปลี่ยนเส้นทางอัตโนมัติเมื่อสถานการณ์เปลี่ยนแนะนำศูนย์พักพิงที่เหมาะสมและช่วยจัดลำดับความสำคัญในการช่วยเหลือผู้ประสบภัย(ที่มา[https://roots.tech/blog/technology-1/what-is-ai-79](https://roots.tech/blog/technology-1/what-is-ai-79))

สรุป**Artificial Intelligence (AI)** คือเทคโนโลยีที่ช่วยวิเคราะห์ข้อมูลและตัดสินใจ เพื่อแนะนำเส้นทางอพยพและการช่วยเหลือที่ปลอดภัยและเหมาะสมที่สุด โดยอาศัยข้อมูลจาก GPS, GIS และข้อมูลสถานการณ์แบบเรียลไทม์
